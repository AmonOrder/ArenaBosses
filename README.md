# ArenaBosses Mod

![Fabric](https://img.shields.io/badge/loader-fabric-fdf2d5?style=flat-square)

ArenaBosses — это Fabric-мод для Minecraft, который добавляет настраиваемую систему боссов и арен. Вы можете задавать границы арены, настраивать спавн боссов и управлять лимитами призывов прямо в игре.

## Основные возможности
* **Система регионов**: Настройка зон арены с защитой блоков.
* **Кастомное меню**: Удобный графический интерфейс (GUI) для вызова боссов.
* **Управление боссами**: Добавление и удаление боссов через команды.
* **Лимиты**: Настраиваемая система ограничений на призыв для игроков.
* **Конфигурация**: Гибкие настройки в `config/arenabosses/`.

## Команды
* `/arena menu` — открыть меню призыва (только внутри арены).
* `/arena clear` — выполнить чистку арены от мусора (блоки и сущности).
* `/arena set <pos1> <pos2>` — задать территорию арены (админ).
* `/arena delete` — удалить территорию арены (админ).
* `/arena spawn set` — установить точку спавна боссов (админ).
* `/arena spawn delete` — удалить точку спавна боссов (админ).
* `/arena boss add <id> <name> <desc>` — добавить босса в список (админ).
* `/arena boss remove <name>` — удалить босса из списка (админ).
* `/arena reset` — сбросить все лимиты призывов и кулдауны (админ).
* `/arena reload` — перезагружает конфиги в мод (админ).
* *(и другие, см. документацию)*

## Установка
1. Установите [Fabric Loader](https://fabricmc.net/).
2. Скачайте файл мода `.jar`.
3. Поместите его в папку `mods` вашего сервера и клиента.
4. Убедитесь, что у вас установлены все зависимости.
